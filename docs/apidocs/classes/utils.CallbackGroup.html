Function static

Defined in src/common/utils.js:307

a syntax sugar for calling other object's methods on an object

Returned function will function just like inherit's $parent, only that if you only pass it arguments it will call the constructor (like inherit#$construct)

Parameters:

  • target Object

    object

  • parent Object

    object

  • [use_directly] Bool optional

    if set to true, will use parent directly, rather than use it's prototype

Returns:

Function:

Example:

this.parent = utils.bindParent(this, Foo);

this.parent([somearg]); //will call Foo - bound to obj this.parent('bar',[somearg]); //will call Foo.prototype.bar - bound to obj

Defined in src/common/utils.js:144

This function waits until all specified font-families loaded and then executes a callback function. Supplied font-families should be already defined in the document, by URL or base64. If after specific threshold time fonts still not loaded, callback will be invoked.

Parameters:

  • fontFamiliesArray Array

    Array of font-families to test

  • fontsLoadedCallback Function

    Callback function to call after all font-families loaded
